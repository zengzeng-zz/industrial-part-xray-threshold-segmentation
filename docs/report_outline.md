# 课程设计报告提纲

## 1. 课题分析与需求分析

- 说明工业零件 X 光检测的应用背景和意义
- 分析原始图像中的噪声、低对比度和缺陷难分离问题
- 明确系统的功能目标、输出结果和评价方式

## 2. 理论基础与算法研究

- 图像灰度化和常见滤波方法
- 阈值分割方法：全局阈值、Otsu、自适应阈值
- 轮廓提取与缺陷区域标注
- 面积、周长等缺陷特征统计指标

## 3. 系统方案设计

- 整体流程：原图输入 -> 预处理 -> 阈值分割 -> 缺陷提取 -> 特征统计 -> 结果输出
- 模块划分与职责
- 参数设置依据与多方案对比思路

## 4. 程序设计与软件实现

- 开发环境与依赖说明
- 各模块主要功能与代码结构
- 关键函数说明与运行流程

## 5. 实验结果与分析

- 原图、预处理图、分割图、标注图展示
- 不同阈值方法的实验对比
- 缺陷统计表结果分析
- 算法优缺点、适用场景和问题讨论

## 6. 问题分析与解决

- 噪声过大导致误检
- 阈值设置不合理导致漏检
- 缺陷区域断裂或粘连
- 结果稳定性与可复现性改进

## 7. 总结与展望

- 完成的工作与主要成果
- 本课题的不足
- 后续可改进方向
