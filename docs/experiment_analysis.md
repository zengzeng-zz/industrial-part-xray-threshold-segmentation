# 实验分析草稿

## 1. 实验目的

验证不同阈值分割方法在工业零件 X 光图像缺陷提取中的效果差异，并结合缺陷面积、周长等特征统计结果，分析算法的适用性。

## 2. 对比方案

- 方案一：全局固定阈值
- 方案二：Otsu 自动阈值
- 方案三：自适应阈值

## 3. 评价维度

- 缺陷区域是否完整
- 背景噪声是否被误识别为缺陷
- 微小缺陷是否能够保留
- 统计结果是否稳定
